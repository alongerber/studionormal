# תיקונים ושדרוגים — כל הסבבים

## סבב 2 (אחרון) — 3 שדרוגים גדולים

### 1. Prompt caching — חיסכון ~90% על טוקני קנון
- כל קריאה שנייה תוך 5 דקות = משלמת 10% מהתעריף הרגיל על הקנון/system prompt.
- אוטומטי, אין להפעיל. Anthropic מטפלים בזה.
- **איך לבדוק:** DevTools → Network → ייצא טיוטה → ב-response.usage תראה `cache_creation_input_tokens` (קריאה ראשונה) ואז `cache_read_input_tokens` (קריאות הבאות).

### 2. Performance v2 — מסך מהיר במקום modal-per-video
- 📈 בכותרת פותח עכשיו תצוגה חדשה (לא modal).
- **טבלה למילוי מהיר:** כל סרטון = שורה. Tab לעבור בין תאים. שדות: תאריך, צפיות, לייקים, תגובות, שיתופים, עוקבים+.
- כפתור **"💾 שמור הכל"** — שומר את כל השורות שמילאת בלחיצה אחת.
- **📋 הדבק טבלה** — מדביק מ-Excel/Sheets/TikTok analytics. פורמט: `כותרת [Tab] צפיות [Tab] לייקים [Tab] תגובות [Tab] שיתופים`. התאמה אוטומטית לכותרת קיימת.
- ⚙ ליד כל שורה — לטופס מלא עם חשיפה/קבוצות זריעה/הערות.
- **תוקן:** היה חסר שדה לייקים בכלל. הניתוחים עד היום היו עם 0 בלייקים.

### 3. לולאת למידה — מה שעבד חוזר ל-prompts
- אחרי 3+ סרטונים עם נתונים → 🧠 **נתח דפוסים**.
- מנתח + בוחר אוטומטית את 3 הסרטונים עם ה-engagement הכי גבוה (`(לייקים+תגובות+שיתופים) / צפיות`).
- שומר אותם כ-`topPerformersBank` ב-DB.
- כשתייצר טיוטה/חלופות/hooks חדשים — הקטעים שלהם מוזרקים אוטומטית כ-YES BANK ל-system prompt.
- **toggle**: "הפעל לולאת למידה" — מופעל כברירת מחדל. אפשר לכבות מהמסך ביצועים אם רוצים brainstorm נקי.

## סבב 1 — 13 באגים שתוקנו

### באגים לוגיים אמיתיים
1. **Canon double-context** — `PROMPTS.canonCheck` הזריק קנון בעצמו + ה-system prompt גם כן. תוקן + casing snake→camel.
2. **crossMode missing context** — המרות בין מצבים לא קיבלו קנון. עכשיו מקבלות.
3. **state.mode race condition** — `convertToMode`/`convertMomentTo` שינו state.mode בזמן async. עכשיו מועבר כפרמטר.
4. **buildContext עם תגיות בקונטקסט** — `[exhales]` נכלל בהקשר. עכשיו מסונן + גודל מותאם למצב (post=1, whatsapp=5, video=3).
5. **dead `ctx` variable + duplicate fullScript ב-fixLine** — תוקן.
6. **Unsaved textarea content** — שינויים נמחקו בשקט במעבר בין שורות. עכשיו: סמן `•` ליד מספר השורה + confirm לפני ניווט + beforeunload warning.
7. **Ctrl+Z swallowed in textarea** — ה-listener חטף את ה-shortcut. עכשיו עובר ל-undo native של הדפדפן.
8. **Stale canon check** — verdict ישן נשאר אחרי שהטקסט השתנה. עכשיו מתאפס בכל edit/delete.
10. **No request cancellation** — לחיצה כפולה = שתי קריאות API במקביל. עכשיו AbortController מבטל את הראשונה.
12. **Variety default = twin (כמעט זהה)** — שונה ל-`similar`. ה-class ב-HTML עבר.
14. **Mark target line in fullScript** — המודל לא ידע איזו שורה להחליף. עכשיו מסומן `← ★ זו השורה להחליף ★`.
16. **freeText נמחק בין שורות** — עכשיו נשמר ב-state ומשוחזר.
17. **buildContext גודל לא מותאם למצב** — תוקן בתיקון #4.

### בונוסים
- **altCount slider** — 1-8 חלופות במקום 5 קבוע.
- **dedup חלופות** — אם המודל מחזיר 2 חלופות שמתחילות אותו דבר, רק אחת מוצגת.

## אזהרות תאימות

**רשומות ביצועים קיימות לא יהיו להן שדה `likes`.** הקוד מטפל ב-`p.likes || 0`. אם רוצים נתונים מדויקים — לפתוח כל רשומה ידנית ולמלא.

**toggle לולאת למידה מופעל כברירת מחדל.** אם אתה לא רוצה שהמודל ילמד עד שתבחר במפורש, לכבות אחרי הניתוח הראשון.

**Cache TTL הוא 5 דקות.** אם עוברים יותר מ-5 דקות בין קריאות, הקנון נטען מחדש (תשלום מלא + 25%). שווה לעבוד ברצף כשאפשר.
